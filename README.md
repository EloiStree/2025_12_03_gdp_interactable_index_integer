# 2025_12_03_gdp_interactable_index_integer
With a bit of abstraction, this allows you to link an IID interaction by dropping a tag and a parent to it.

“It’s incredibly boring to manually link each button to an event that sends an integer when you’ve got hundreds of them. It’s much easier to just drag-and-drop a config script and assign a parent on top.
